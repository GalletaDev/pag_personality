## 🚀 ¿Qué incluye?

- 📄 **HTML**: La estructura básica de la página (como los huesos del cuerpo).

## 📸 Uso de Cloudinary (opcional)

Use cloudinary para ahorrar recursos y no enviar cada imagen en la pagina
asi optimizas y sera mas rápido en cargar recursos.


(Recomendación): utiliza esto si tienes imágenes no muy pesados.
como logo de marca o imágenes .webp o .avif ya que son los menos pesados.
es lo mejor ya que dependiendo de cuanta gente mira tu pagina, el consumo es mas pesado.
ya que consumes créditos cada vez que miran la imagen o descarguen.


¿Se paga esto?
-No, es completamente gratis, pero solo date cuenta que tiene un limite de 25 GB o créditos de uso.
pero si pensamos bien, mientras las imágenes no pesen mucho, puedes ser suficiente

1. Crea una cuenta gratuita en [https://cloudinary.com](https://cloudinary.com).
2. Sube tus fotos.
3. Copia el enlace de la imagen que te da Cloudinary.
4. Pega el enlace en el HTML así:
```html
<img src="https://res.cloudinary.com/tu-usuario/image/upload/v1234567890/foto.jpg" alt="Foto personal" />
```

tiene disponible usar embed para tu paginas, asi que úsalo para tener mas espacio debes de tu pc
